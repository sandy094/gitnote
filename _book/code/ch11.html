TypeScript/Angular

Angular/TypeScript

Angular CLI指令列工具
1.安裝Angular CLI並產生Angular專案 

npm install -g angular-cli

angular相關命令行皆用ng
常用指令
ng new [project name]: 建立新專案，同時也將相關套件載下來記錄在package.json。

ng generate(ng g): 用來產生Angular的元件(component).服務(service)等程式雛形。

EX:
ng g compoent new-component(ng g c new-component)
ng g directive new-directive
ng g service new-service
