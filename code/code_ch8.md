## Python
### 列表

新增
```
names:["A,B,C,D"]
<!-- 新增在後面 -->
names.append("E")
<!-- 新增在特定一個位置 -->
names.insert(1,"F")

print(names)
<!-- 執行結果 -->
["A,B,C,D,E"]
["A,B,C,D,E"]
["A,F,C,D,E"]

```

刪除
```
names:["A,B,C,D"]
<!-- 沒特別說明就是刪除最後一個 -->
names.pop()


```
